My Web Shorcuts is a free and open source browser extension that lets you create custom keyboard shortcuts to click (or do other actions) on elements in any website.

Frequently Asked Questions

First of all, what is My Web Shortcuts?
Good First Question!
My Web Shortcuts is a browser extension built for clicking (& more) on elements in a website using keyboard keys. It let's you select any element (button, link, etc.) on a website, and asks you for the shortcut key you want to use to click on it. Now, whenever you click on the key, that element will be clicked.
What are some possible use cases of this?
Well, it depends on you and the websites you visit. We aren't really sure how many possible use cases there can be, but it's certain that navigating and accessing the web will be much faster with this tool.
Most of the times, you will be creating shortcuts for the websites you often go to, & elements you often click on. Previously you used to drag your cursor every time to click on it, but now, you will simply press the shortcut key.
Is it paid?
My Web Shortcuts is 100% free & open source. We do need to your support to keep working on it, so if you are capable, please considering supporting us.
Can I sync the shortcuts on my other browsers and computers?
We're currently in the beginning of it's development, so syncing feature is not available. However, we will be adding an option to import/export shortcuts.
Please note that the syncing feature will require us to pay for the recurring cloud bills, and therefore, we would need to make that feature a paid one. If we're able to get a lot of support from the community, we can make it free upto a limit.
I'm a web developer & interested in contributing to the project.
We're pleased to know that! Please go to github.com/mywebshortcuts to know how you can contribute.
